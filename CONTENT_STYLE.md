# Plain-language content style

Write for a Burmese-speaking household helper who may need to act quickly.

## Write short instructions

- Use headings of 2–5 words.
- Use button labels of 1–3 words.
- Put 1 action in each step.
- Start steps with a verb.
- Put the most important action first.
- Use digits: `2 weeks`, `100 g`, `1 cup`, `10 min`, `190°C`.
- Use the same word for the same object. Use “fridge,” not “refrigerator.”
- Keep safety warnings complete and direct.
- Do not repeat the summary as a step.

## Burmese first

- Use natural, everyday Burmese.
- Prefer short spoken-style sentences.
- Avoid long noun phrases and formal wording when a common word works.
- Keep useful names such as Nako, Edwin, WhatsApp, Royal Canin, and Firebase unchanged.
- Use an English product or button name only when the helper is likely to see that exact word.
- Never copy English into the Burmese field as a fake translation.
- Check that Burmese remains valid Unicode.

## English and Japanese

- Keep the same meaning as the Burmese instruction.
- Use direct, respectful language.
- Keep measurements, names, times, temperatures, medicine details, storage limits, and emergency actions exact.

## Manual review checklist

- Can the main action be understood in one quick read?
- Does each line ask for only one action?
- Is any safety detail missing?
- Are EN, JP, and MM complete and consistent?
- Is any heading, label, or summary longer than needed?
- Is the same sentence repeated elsewhere on the page?
